Ik ben!

ik wil me even voor stellen, ik ben nanco lin en ik ben nep chinees. Ik heb een vooropleiding grafisch ontwerp gedaan on het Deltion te Zwolle. Ik ben 23 jaar en mijn interesse zijn gamen, anime en technologie. Op dit moment zit ik op het saxion ter Enschede, en volg de opleiding van kunst en techniek.

Een grootte inspiratie bron voor de Chinese klok was de nixie tube klok. Ik wil al lange tijd een kopen en met dit project heb ik een goed excuus om er een te bouwen voor mij zelf. Ik heb niet precies een nixie tube klok gemaakt, ik heb er mijn eigen creatieftijd heen gegooid waar door het een product van nanco lin is geworden!

Mijn eerste idee

Wat ik wil maken is een digitale klok met Chinese karakters als vervanging voor de cijfers in de klok. Het idee kwam van een anime die ik heb gekeken (Monogatari series).

De rede waarom ik heb gekozen om dit maken is, het is een reflectie van wie ik ben als persoon. Als ik mijn zelf moet omschrijven wie ik ben, dan zal ik zeggen dat ik een nep Chinese ben. Dit zegt veel over mijn, mijn karakter en wie ik ben. Ik ben iemand die is opgegroeid met twee culturen. Hier door heb ik de normen en waardes van de Chinese cultuur en de normen en warden van de nuchtere Nederlander. En dat wil ik terug laten komen in mijn ondwerp van de klok.

Het eerste idee voor de klok is een standalone klok, dat betekent geen computer of ander rand apparatuur. Hier voor heb ik veel onderzoek over gedaan of het mogelijk is. Volgens mijn onderzoeken is het mogelijk en ik heb de nodige dingen aan geschaft om het te maken. Naar wat tijd ben ik toch er achter gekomen dat het te hoog gegrepen is voor mijn. Want de codering van de klok die ik heb geschreven moet ik omzetten in c++. Dit is bijna onmogelijk. Daarom ben ik terug gevallen op de back up plan van een laptop. Hier door hoef ik de code die ik heb geschreven niet over te zeten. Door deze besluit werd mijn plan voor een standalone klok dramatisch verandert naar iets heel anders.

Hoe ik ben begonnen

Ik ben begonnen met het schrijven van de code voor de klok. De code was er voor om de nummers te veranderen naar Chinese karakters. Dit heb ik gedaan met java. Later heb ik de code geport naar processing. Op dit moment kwam ik er ook achter dat veel van mijn idee niet gaan werken. Één van de ideeën was een standalone klok, dat idee was niet mogelijk. De code die ik heb geschreven was niet tot zeer moeilijk te porten zijn maar Arduino. Daarom heb ik besloten er een monitor er bij te houden. Zo kan ik de code die ik heb geschreven toch gebruiken. Een van de dingen die we samen hebben geschreven is de header van de website.

Werken aan het project

Het meeste werk voor het project heb ik samen gedaan met een vriend van mijn (Jason Allon). Wij hebben het meeste werk samen gedaan over teamviewer en skype. We met elkaar overleggen en elkaar geholpen om er achter te komen hoe we het moeten coderen. Op deze manier hebben wij alle twee nieuwe dingen geleert. Zo is mijn kennis van java met sprongen vooruit gegaan.

Verder heb ik een goede plan gemaakt aan het begin van het kwartiel. Hier staat in wat ik wil en wat ik al kan. Hier me kon ik zien waar mijn probleem punten liggen.

Met het morfologisch schema heb ik ook een goed vangnet voor als dingen fout gaan. Dit schema heb ik ook meerder keren gebruikt.

Het eind product

De klok is heef anders geworden dan wat ik zelf in gedachte had. Het is nu een klok geworden die op de laptop woord getoond in plaats van een standalone klok. Deze verandering vindt jammer. Ik wouw iets maken wat een soort van kunst werkje op zich was. Dit is helaas niet gelukt met de kennis die ik nu bezit.

Ik heb een besturing doos bij de klok. Op de doos zitten meerder knopen waar je de klok me kan bedienen. En van de knopen is er voor on de klokte kunnen uit lezen. En er zijn ook ander knoppen om andere dingen te kunnen veranderen.

Nanco Lin (C.Clock)
